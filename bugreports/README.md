# Bug Reports

This folder is reserved for public-facing bug report material when that flow is modeled later.

Current direction:

- this folder is intended to hold bug-report screenshot images
- those images are expected to come from the future in-app `Submit Bug` flow
- uploaded screenshots should later pass through a metadata-strip/security step before handoff
- that future security step may rename the image files as part of sanitizing them
- the future bug-report model is intended to hand the report off to Discord
- optional player-submitted log files may also be included in that future Discord handoff
- GitHub Issues are not intended to be the bug-report intake path for this flow
- bug reports for this modeled lane should go to Discord only

Current rule:

- do not treat this as active development space
- use it only for public/release-side bug report items if that process is added later
